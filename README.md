# Spros Half-Swept
A custom version of the [Sweep](https://github.com/davidphilipbarr/Sweep).

| Front | Back |
| --- | --- |
| ![](/img/front.png) | ![](/img/back.png) |

Spros Half-Swept is a split keyboard based on the Ferris Sweep Half-Swept, which, in turn, is half of the Sweep2 modified for a reversible microcontroller footprint.

The goal of this build is to create a compact, low-profile keyboard that offers:
- Affordability
- Stylish design
- Comfort
- Practicality
- Hotswap
- Easy portability
- Bluetooth connectivity
