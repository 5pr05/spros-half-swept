# Spros Half-Swept
A custom version of the [Sweep](https://github.com/davidphilipbarr/Sweep).

| Front | Back |
| --- | --- |
| ![](/img/front.png) | ![](/img/back.png) |

Spros Half-Swept is a split keyboard based on the Ferris Sweep Half-Swept, which, in turn, is half of the Sweep2 modified for a reversible microcontroller footprint.

[Spros Half-Swept Case "bl0.x"](https://www.printables.com/model/1087752-c453)
[Spros Half-Swept Case "c453"]

## 5PR05 53TUP:

| Parts ordered from `Aliexpress`                         | Quantity |
| ------------------------------------------------------- | -------- |
| NRF52840 Controller                                     | 2 pcs    |
| Li-Po battery 30x35x5mm (or smaller)                    | 2 pcs    |
| JST-XH Battery Connector Male and Female                | 2 pcs    |
| M2 insernut (7mm len, 3.18mm D2)                        | 10 pcs   |
| M2 screws with small head (8mm len)                     | 10 pcs   |
| Neodymium Magnets 8x2mm                                 | 10 pcs   |
| Kailh Choc Switches                                     | 34 pcs   |
| Blank MBK Choc Keycaps 1u                               | 34 pcs   |
| Reset Buttons b3u, on/off switch msk12c02               | 2 pcs    |
| TRRS jack                                               | 2 pcs    |
| Folding laptop feet                                     | 2 pcs    |



The goal of this build is to create a compact, low-profile keyboard that offers:
- Affordability
- Stylish design
- Comfort
- Practicality
- Hotswap
- Easy portability
- Dual connectivity (Bluetooth and wired options)
